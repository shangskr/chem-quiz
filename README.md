# 化工总控工（高级）答题系统

基于官方题库（1841 题）的在线答题系统，支持刷题、顺序练习、模拟考试三种模式，**多设备进度同步**。

## 功能

- **刷题模式** — 随机抽题，即时反馈对错
- **顺序模式** — 从指定题号连续往后，即时反馈
- **考试模式** — 全部答完才显示成绩，仿真机考，自动倒计时
- **错题本** — 标记重点题目，集中复习
- **错题作答** — 结果页一键重做错题
- **多设备同步** — 输入口令即可跨设备同步答题进度（基于 MongoDB）
- **学习统计** — 三个模式独立统计（判断/单选/多选），口令用户可上传云端
- **本地模式** — 不输入口令也可答题，数据保存在本地浏览器
- **夜间模式** — 手动切换或自动跟随系统主题
- **管理员面板** — 查看所有用户数据（进度/标记/统计），支持选择性删除
- **键盘快捷键** — ← → 切换题目，Enter 确认答案
- PC 和手机均适配

## 数据存储说明

| | 普通用户 | 口令用户 |
|---|---|---|
| 答题进度 | localStorage | localStorage + 云端同步 |
| 学习统计 | localStorage | localStorage + 云端同步 |
| 错题标记 | localStorage | localStorage + 云端同步 |
| 主题/字号 | localStorage | localStorage |

- **清除缓存**：仅清除本地数据（进度+统计+主题字号），保留错题标记，不影响云端
- **清除标记**：清除错题标记，口令用户同步清除云端
- **清除云端统计**：统计面板下方按钮，仅口令用户可见，删除云端统计

## 部署到 Vercel

1. Fork 或推送本项目到 GitHub
2. 在 [vercel.com](https://vercel.com) 导入该仓库
3. 在 Vercel 项目左侧 **Settings → Environment Variables** 添加：
   - `MONGODB_URI` — MongoDB 连接串
   - `ADMIN_KEY` — 管理员密钥，用于登录管理员面板
4. 部署完成即可使用

## 使用说明

1. 打开部署后的网页
2. 在首页输入**同步口令**（任意字符串），点击"加载"；或不输入口令直接答题（数据仅存本地）
3. 同一口令在不同设备上登录，共享同一份进度数据
4. 答题进度在**退出或交卷**时同步到云端，答题过程中每选一题自动保存到本地

## 本地运行

直接用浏览器打开 `index.html` 即可，但同步功能和管理员面板需要部署到 Vercel 才能使用。

## 题库格式

`questions.js` 中导出一个 `const QUESTIONS` 数组，每道题结构：

```js
{
  id: 1,                    // 题号
  type: '判断题',            // 判断题 | 单选题 | 多选题
  text: '题目内容',
  options: [                // 选项列表
    { label: 'A', text: '选项内容' }
  ],
  answer: 'A',              // 正确答案（多选题用分号分隔，如 "A;B;C"）
  score: 1                  // 分值
}
```

替换题库只需生成同结构的 `questions.js`，前端无需修改。
